Find out the number of quotes

http://localhost:3000/quote/total 

      
        "Total number of quotes: 12"

Routes

Endpoints Description Example
GET/quotes return all quotes http://localhost:3000/quotes
GET/quote/random return a random quote http://localhost:3000/quote/random
GET/quote/id/:id return quote by its ID http://localhost:3000/quote/id/5
GET/quote/member/:member return an array of quotes by member http://localhost:3000/quote/member/Дэймон Сальваторе
GET/quote/total return total number of quotes http://localhost:3000/quote/total
